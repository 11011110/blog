* The [long-predicted cleveref apocalypse]({{site.baseurl}}{% post_url 2025-09-15-linkage %}) is upon me <span style="white-space:nowrap">([$$\mathbb{M}$$](https://mathstodon.xyz/@11011110/116349463567154866)).</span> For the most part it works to just replace 

  ```
  \usepackage[capitalize,nameinlink]{cleveref}
  ```

  by

  ```
  \usepackage{zref-clever}
  \zcsetup{cap}
  \let\cref\zcref
  ```

  I have been using an "observation" theorem type, and for this I need to tell zref-clever what to call them:

  ```
  \zcRefTypeSetup{observation}{Name-sg=Observation,Name-pl=Observations}
  ```

  If you want to have different theorem types share the same numbers, things get more complicated. I haven't needed that yet, but there's guidance in Section 10.2 of [the zref-clever user manual](https://mirrors.ctan.org/macros/latex/contrib/zref-clever/zref-clever-doc.pdf).
